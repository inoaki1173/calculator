# Calculator

Vue学習＆就職活動のポートフォリオ用に作成した電卓アプリです。
シンプルな二項演算を実装しています。

## 操作方法

各ボタンをクリック、または対応するキーボードのキーを押すことで操作を行える

### キー対応表

| キーボード  |            対応操作            |
| :---------: | :----------------------------: |
|    0～9     |              数字              |
|     ．      |             小数点             |
| +　-　\*　/ |             演算子             |
|      n      |            符号反転            |
|    Enter    |            計算実行            |
|  BackSpace  |   入力中の数字を<br>1桁消去    |
|   Delete    |  入力中の数字を<br>すべて消去  |
|   Escape    | すべての数字と<br>計算式を消去 |
